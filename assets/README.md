# Assets Folder

This folder contains static assets for the TECTONIQ application.

## Logo

**File:** `tectoniq_logo.png`

The TECTONIQ logo featuring a seismograph on parchment paper with the TECTONIQ branding.

### Usage:

- Login/auth page (200px width)

### Instructions:
Please save the TECTONIQ logo image as `tectoniq_logo.png` in this folder.

The application will automatically use it if present, or fall back to a placeholder if not found.
